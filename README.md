
# Installation

 1. Open Finder window at `Users/USER` level
 2. Hit `Cmd` + `Shift` + `.` (to show hidden files/folders)
 3. Open `.vscode` folder > `extensions`
 4. Paste `daze.theme` folder into here
 5. Restart VSCode
 6. In VSCode, on menu bar, go to `Code` > `Preferences` > `Color Themes`
